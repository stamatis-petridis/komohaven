# Sync Language Variants

Review and synchronize all four language variants of the landing page.

## Your Task

1. Read all four landing page files:
   - `index.html` (English - canonical)
   - `index-gr.html` (Greek)
   - `index-tr.html` (Turkish)
   - `index-bg.html` (Bulgarian)

2. Compare the DOM structure of each file against `index.html`:
   - Check that all section IDs match
   - Verify all data attributes are present (`data-contact`, `data-map-link`, `data-prefill-rental`)
   - Confirm CSS classes are identical
   - Validate hreflang tags reference all four variants

3. Report any structural differences found

4. If requested, fix the discrepancies by updating the non-English files to match the English structure while preserving translated text content
